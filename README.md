## Getting Started

The package management tool for this project is [Yarn](https://yarnpkg.com):

```bash
yarn install
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


### This project uses:
- Typescript
- React
- Next.js
- Axios
- @volvo-cars/css
- @volvo-cars/favicons
- @volvo-cars/react-icons
- canvas-confetti
